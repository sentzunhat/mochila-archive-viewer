# 007: Match POC UI structure in new Go implementation

**Status:** done
**Created:** 2026-07-09
**Closed:** 2026-07-09 (backtracked — implemented before HAWP tracking was established)

---

## Context

The original POC (`archive/snapchat-export/app/src/App.svelte`) has a clean, proven UI layout. The new Go implementation (`src/frontend/src/App.svelte`) adds multi-user profile system, Wails RPC bindings, and progressive image loading — but some visual details and behaviors drifted from the POC during porting. This work item audits and fixes those gaps while preserving all new backend capabilities.

---

## Scope: Verified Gaps

### 1. Year sidebar bar chart uses wrong denominator

**POC:** `width:${Math.max(4, (count / maxYearCount) * 100)}%` where `maxYearCount = Math.max(...years.map(([, count]) => count), 1)`  
**New:** `width:${Math.max(4, (count / summary.mediaCount) * 0.5)}%` — divides by total media count instead of max year count, making bars too thin and inaccurate.

### 2. Category and Type filters don't reset pagination

**POC:** All four filters (`on:change={() => (visibleLimit = 180)}`)  
**New:** Only Year calls `setYear(selectedYear)` which resets visibleLimit. Category and Type selects lack any handler, so filtering by category/type doesn't reset the visible page.

### 3. Video tiles missing controls attribute

**POC:** `<video preload="metadata" controls muted src="/media/${item.id}">`  
**New:** `<video preload="metadata" muted src={sourceFor(item)} playsinline>` — no `controls`, user can't pause/seek preview videos in the gallery grid.

### 4. Media modal missing Escape key handler

**POC:** Has `handleModalKeydown` listener that sets `selectedMedia = null` on Escape.  
**New:** No escape handler on the media modal overlay — user must click the × button.

### 5. JSON preview rendering

**POC:** Renders `JSON.stringify(selectedJson.summary.sample, null, 2)` in a `<pre>` block with child counts grid.  
**New:** Has `jsonPreview` state and `GetJSONPreview` Wails binding but may not render the same detail layout (needs verification of template section).

### 6. Structure view layout

**POC:** Simple zip grid + raw summary JSON at bottom.  
**New:** Uses tabs (Paths / Archives / Summary) — different UX, not necessarily worse, but worth documenting as intentional deviation.

---

## Intentional Deviations (Not Bugs)

| POC Feature                                        | New Behavior                                | Rationale                                                                        |
| -------------------------------------------------- | ------------------------------------------- | -------------------------------------------------------------------------------- |
| `/media/${id}` direct URLs                         | Wails `GetMediaSource(platform, id)` RPC    | Go backend serves from local disk via HTTP handler instead of static file server |
| Eager conversation load (`fetch("/api/messages")`) | Lazy load per-conversation via Wails RPC    | Avoids loading all messages at startup; better for large exports                 |
| No image loading states                            | Placeholder div while fetching media source | Progressive enhancement — shows "Loading…" before Wails returns the path         |
| Single flat structure view                         | Tabbed (Paths/Archives/Summary)             | Better organization for multi-platform archives                                  |
| No profile system                                  | Multi-user profile with login/logout        | New feature, no POC equivalent                                                   |

## Risks

- Adding `controls` to videos may change grid height dynamics — videos with controls bar are taller than muted thumbnails.
- The tabbed structure view is a deliberate UX improvement over the flat POC layout; changing it would reduce work for this item.

## Verification

- Year sidebar bars render proportional to max year, not total media count
- Selecting Category or Type filter immediately resets visibleLimit to 180
- Gallery video tiles show play/pause controls
- Pressing Escape closes the media modal
- Data Explorer renders JSON preview with child counts and sample data
