### Commit 1 ‚Äî `fix(020): safety/correctness + dead code removal` ‚Äî `f9fa604`

**Safety / data-loss (A1, A4, A5, C1, G1):**
- **A1**: `platform_snapshots` PK changed to `PRIMARY KEY(platform, user_id)` via SQLite table-recreation migration. `ON CONFLICT(platform, user_id)`. Prevents second user's `IndexArchives` from overwriting first user's counts.
- **A4**: `migrate()` profile DDL propagates errors for create/copy/drop/rename steps.
- **A5**: `PlatformStats` propagates all 6 scan errors via a local helper; no silent zeros.
- **C1**: Removed duplicate `s.store.db.Query` UNION ALL block from `GetPlatformStats` ‚Äî layer boundary violation + redundant DB queries.
- **G1**: `ServeHTTP` returns HTTP 500 for service errors vs HTTP 404 for genuinely missing items.

**Dead code removal (B1‚ÄìB3):**
- **B1**: Removed `cacheMediaFiles` + `mediaCachePath` from `service.go`. Call removed from `IndexArchives`. The HTTP handler reads from zips directly; disk cache was written but never read since 019.
- **B2**: `MediaItem.LocalPath` changed to `json:"-"`.
- **B3**: `GetMedia/GetConversations/JSONFiles/GetConversation/JSONPreview` in-memory fast-paths removed. All reads go through the Store.

**Type cleanup (D1, D3):**
- **D1**: `UserEntry` type removed; `AvailableUsers` returns `[]Profile` in store, service, and appshell.
- **D3**: `AppSettings.Homedir` gains `json:"homedir"` tag; no-op self-assignment removed from `GetAppSettings`.

**Schema indexes (N1):**
- Indexes added on `(platform, user_id, year)`, `(platform, user_id, type)`, `(platform, user_id, category)`, and `profile(logged_in)` via `migrate()`.

**`PlatformState` + `Snapshot` slimmed** ‚Äî only `Selected + Summary` restored at startup; heavy data always fetched from DB on demand.

### Commit 2 ‚Äî `docs(020): add package-level GoDoc to types, archive, and appshell` ‚Äî `7cb7e99`

- **E1**: Package-level GoDoc added to `types`, `archive` (via `storage.go`), and `appshell`.
- **E2**: All exported `Store` methods documented in the `storage.go` rewrite.
- **E4**: `migrate()` now has Phase 1‚Äì5 section labels.
- **A2**: `loadGroupedCounts` rewritten with explicit query constants (no string interpolation).
- **A3**: `MediaPaginated` LIMIT/OFFSET uses `?` placeholders.
- **H3**: `ConversationByID` added ‚Äî direct `WHERE conversation_id = ?` query; `GetConversation` in service routes to it.
