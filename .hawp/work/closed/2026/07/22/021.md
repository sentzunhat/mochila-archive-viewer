## What Was Done

- **G2 (IsSender):** `readOwnerName()` reads `personal_information.json` from two candidate paths; `IsSender: ownerName != "" && m.SenderName == ownerName`
- **G3 (media linking):** Two-pass indexer — collect media in first pass, build `entryToken → MediaItem.ID` map, parse threads in second pass using the map
- **G4 (stable ID):** `thread.ThreadPath` used as conversation ID; fallback to `filepath.Base(filepath.Dir(entry)) + "/" + filepath.Base(entry)`
- **G5 (service wiring):** `service.platform()` allows `"instagram"`; `IndexArchives` dispatches `instagram.IndexZips`; `MediaBytesForUser` dispatches `instagram.ReadEntry`; `ProviderCards` marks instagram supported
